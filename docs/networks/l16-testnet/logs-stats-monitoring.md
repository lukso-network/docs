# Logs, Stats and Monitoring

## Execution logs

```sh
sudo lukso network log execution -f
```

You can close the logs by pressing `ctrl+c`

## Consensus logs

```
sudo lukso network log consensus -f
```

You can close the logs by pressing `ctrl+c`
